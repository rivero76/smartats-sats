# Documentation Hub

This folder contains product, technical, help, release, and compliance documentation.

## Sections
- `templates/`: reusable document templates
- `specs/`: product and technical specifications
- `help/`: user documentation and troubleshooting
- `releases/`: release notes by year/month
- `changelog/`: global product changelog
- `runbooks/`: operational procedures
- `compliance/`: data/privacy and audit policies
